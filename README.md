# 环境配置
`VS Code`下安装`Go`插件后，提示需要安装一些工具，但是在墙内会经常导致失败。可以[在此](https://pan.baidu.com/s/1mjLPKfe)下载已经编译好的`Windows`版本。将这些二进制文件放置到`GOPATH\bin`目录下即可。

# 运行
> go run main.go

# 删除临时文件
> go clean -i
